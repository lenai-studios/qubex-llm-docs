## Installation Overview

Qubex LLM Desktop is a "**single-player**" application you can install on any Mac, Windows, or Linux operating system and get local LLMs, RAG, and Agents with little to zero configuration and full privacy.

## Docker vs Desktop Version

Qubex LLM offers two main ways to use Qubex LLM. There are some distinct differences in functionality between each offering. Both are open source.

### You want Qubex LLM Desktop if...

- You want a one-click installable app to use local LLMs, RAG, and Agents locally
- You do not need multi-user support
- Everything needs to stay only on your device
- You do not need to "publish" anything to the public internet. Eg: Chat widget for website

### You want Qubex LLM Docker if...

- You need an easy setup, but server-based service for Qubex LLM to use local LLMs, RAG, and Agents locally
- You want to run an Qubex LLM instance that many people can use at the same time
- You want to be able to share information with our users on your instance you invite
- You need admin and rule-based access for workspaces and documents.
- You will publish chat widgets to the public internet
- You want to access Qubex LLM from the browser
